# Module 02 — Your First API Call

First real API calls. By the end of this module you can send any text to Claude and receive a response.

## Setup

Make sure you've copied `.env.example` → `.env` and added your `ANTHROPIC_API_KEY`.

## Lessons

| File | Lesson |
|------|--------|
| `01_hello_claude.py` | The simplest possible API call |
| `02_request_anatomy.py` | Every field in a request explained |
| `03_reading_the_response.py` | Parsing the response object |
| `04_error_handling.py` | Handling rate limits, timeouts, and API errors |

## Run

```bash
python 01_hello_claude.py
python 02_request_anatomy.py
python 03_reading_the_response.py
python 04_error_handling.py
```
